--- layout: post title: Текстові редактори в Linux та кодування windows-1251. category: [LINUX] ---

За замовчуванням у linux-системах графічні текстові редактори не визначають кодування windows-1251 автоматично. Як це змінити, на прикладі gedit, xed. Перш за все треба встановити dconf-editor

sudo apt install dconf-editor

Налаштування кодувань знаходяться в ключах /org/gnome/gedit/preferences/encodings/auto-detected, для gedit 3, та /org/x/editor/preferences/encodings/auto-datedted для xed. До ключа треба руками додати значення WINDOWS-1251.
Або через термінал

gsettings set org.gnome.gedit.preferences.encodings auto-detected "['UTF-8', 'WINDOWS-1251', 'KOI8-U', 'CP866', 'CURRENT', 'ISO-8859-15', 'UTF-16']" gsettings set org.x.editor.preferences.encodings auto-detected "['UTF-8', 'WINDOWS-1251', 'KOI8-U', 'CP866', 'CURRENT', 'ISO-8859-15', 'UTF-16']"
