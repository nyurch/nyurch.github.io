Організація доступу до файлів на комп'ютері під керуванням операційної системи Ubuntu за допомогою samba. Samba - реалізація мережевого протоколу SBM/CIFS, що використовується для надання доступу до файлів, принтерів, портів та реалізації іншої взаємодії між елементами комп'ютерних мереж.

  1. Першим кроком є встановлення самого пакету: 
    sudo apt-get install samba
  2. Додаємо у файл /etc/shells запис /bin/false
  3. Створюємо користувача для доступу до спільних файлів: 
    sudo useradd samba -s /bin/false
sudo smbpasswd -a samba
    вводимо для нього пароль з яким він матиме доступ до файлів.
  4. Далі потрібно відкорегувати вміст файлу /etc/samba/smb.conf з урахуванням параметрів мережі та доступу до необхідних ресурсів.

Можна використати наступний файл конфігурації. Він налаштований для ПК що знаходиться в робочій групі Workgroup, має ip в одній з підмереж 192.168.0.0, користувач для авторизованого доступу - samba, та теки до яких надається доступ /home/luna/data та /home/luna/temp. Для анонімного доступу до вказаних ресурсів потрібно розкоментувати закоментовані рядки та закоментувати рядки security = USER та valid users samba. 

[global]
  workgroup = workgroup
   server string = %h server (Samba Server)
;   guest account = nobody
   log file = /var/log/samba-log.%m
   max log size = 1000
   syslog = 0
   lock directory = /var/lock/samba
;   share modes = yes
   interfaces = 192.168.0.0/24 eth0
;   security = SHARE
   security = USER
encrypt passwords = true
follow symlinks = yes
wide links = yes
unix extensions = no

[data]
   comment = Software space
   path = /home/luna/data
   valid users samba
   read only = yes
;   public = yes
;   guest ok = Yes
;   guest only = Yes


[tmp]
   comment = Temporary file space
   path = /home/luna/temp
   read only = no
   valid users samba
;   public = yes
;   guest ok = Yes
;   guest only = Yes
