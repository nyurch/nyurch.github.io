Шпаргалка для нубів :)

  1. Налаштування проксі cd ~ touch .gitconfig git config --global http.proxy http://proxyuser:proxypass@proxyaddress:8080
  2. Клонування репозиторію на ПК git clone https://github.com/nyurch/nyurch.github.io
  3. Заливка змін на github cd /директорія_ропозиторію git add --all git commit -m "$(date +%x)" git push -u origin master
  4. Або якщо не проходить з ключем -u git push -f origin master
  5. Виключити заливку тимчасових, допоміжних файлів та всякого хламу можна створивши в директорії з репозиторієм файл .gitignore та вказати там що потрібно ігнорувати.
